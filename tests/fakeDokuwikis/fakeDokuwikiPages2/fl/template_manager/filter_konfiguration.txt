====== Filterkonfiguration im FINDOLOGIC Kundenaccount ======

Die Filterkonfiguration im FINDOLOGIC Kundenaccount ermöglicht die benutzerfreundliche und schnelle Konfiguration Ihrer Filter. Sie erreichen die [[https://secure.findologic.com/login/index.php?page=filters|Filterkonfiguration]] im Kundenbereich unter Einstellungen -> Filterkonfiguration.
Wählen Sie über Auswahl/den Reiter zunächst ob sie die Filter der Suchergebnisse oder der Navigation konfigurieren möchten. Suchergebnisfilter erscheinen bei Ausgabe einer Suchergebnisseite, Filter der Navigtion erscheinen sofern aktiv bei Aufruf einer Kategorienseite über die Shopnavigation.

===== Kompatibilität =====

Beachten Sie folgende Punkte, um die Filterkonfiguration verwenden zu können.

=== XML Filter===
Sollten Sie eines der Shopsysteme
  * OXID
  * Shopware
  * Magento
verwenden, ist folgendes zu beachten: 

Die Filterdaten werden mittels XML Rückgabe an das Modul des Shopsystems übertragen. Dieses nutzt die im Shopsystem vorhandenen Filterfunktionen um diese in HTML umzuwandeln. Eventuell werden vom Modul nicht alle Filtertypen vollständig unterstützt die im FINDOLOGIC Kundenaccount angezeigt werden. FINDOLOGIC empfiehlt das eingesetzte Modul auf die neueste Version upzudaten, sollten Filterfunktionen vermisst werden.
Die Reihenfolge und Filterart muss zudem über das entsprechende Template des Shops angepasst werden.

=== HTML Filter - Templates ===
Um die Filterkonfiguration verwenden zu können muss das "generated Filters" Plugin aktiviert sein. Das Plugin wird aktiv, sobald nachfolgende Zeile Code in Ihrem Template (Beispielsweise: plenty2.tpl, html2a.tpl) vorhanden ist. Bitte prüfen Sie im FINDOLOGIC Kundenaccount unter [[https://secure.findologic.com/login.symfony/web/app.php/template/index|Templatemanager]] ob dieser Code vorkommt und tragen Sie diesen allenfalls nach.
<code smarty>
{generated_filters}
</code>

Dieses Plugin gibt die im Kundenbereich konfigurierten Filter aus. 

Um Filter nicht doppelt anzeigen zu lassen, sollten Sie bei Aktivierung der Filterkonfiguration die Einträge zu anderen eventuell dargestellten Filtern im Template entfernen.

===== Hauptfilter / weitere Filter / inaktive Filter =====

Filter sind per Drag&Drop (Verschieben mittels Maus) in die Kategorien Hauptfilter, weitere Filter oder inaktive Filter verschiebbar.

=== Hauptfilter ===
Filter in dieser Kategorie werden standardmäßig bei jedem Suchergebnis angezeigt. Sind keine Filterdaten vorhanden, so erscheint standardmäßig eine Filterbox mit dem Hinweis dass keine Filtermöglichkeit vorhanden ist.

=== weitere Filter === 
Filter in dieser Kategorie werden im Shop erst durch einen Klick auf den Link "weiter Filter anzeigen" angezeigt. Sind keine Filterdaten vorhanden, so werden die Filter und der Link nicht angezeigt.

=== inaktive Filter ===
Filter in dieser Kategorie werden immer ausgeblendet.

===== Filter konfigurieren =====
Um alle verfügbaren Einstellungen einsehen zu können klicken Sie den Button "Weitere Einstellungen"

=== Datenbank-Name ===
Der Name des Feldes wie im Export übergeben
=== Anzeige-Name ===
Der angezeigte Name des Filters im Suchergebnis oder Navigation.

Dieser wird für XML belieferte Shopsysteme **NICHT** mitgeliefert.

=== Typ ===
Wählen Sie im Dropdown Feld die Art des Filters. Je nach hinterlegter Datenbasis können Sie aus unterschiedlichen Typen wählen.

== Text ==
Filterwerte werden als verlinkter Text angezeigt.

== Farbfilter ==
Filterwerte werden als Farbkacheln dargestellt.

|{{:fl:farbfiltervn2.png|Farbfilter Vorher/Nachher }}|

Damit alle Farben als Kacheln dargestellt werden können, müssen gegebenenfalls noch Farben im Konfigurator nachgetragen werden. Eine Anleitung hierzu finden Sie unter: |[[fl:farbfilter#Einstellen_der_Farben_im_Kundenaccount|Einstellen der Farben im Kundenaccount]]|

== Dropdown ==
Filterwerte werden in einem Dropdown Element dargestellt. 
|{{:fl:filterkonfiguration_drop_fix.png| }}|

  * Fixierte Filterwerte gibt an, wieviele Filterwerte als Textfilter über dem Dropdown Filter angezeigt werden sollen. Im Dropdown Filter werden dann die restlichen möglichen Filterwerte gelistet. Mögliche Werte: Ganze Zahlen > 0

== Bild ==
Filterwerte werden mit hinterlegten Bildern dargestellt. 
Die Filterdarstellung mit Bildern eignet sich vor allem für Herstellerlogos und wird deswegen standardmäßig nur für den Herstellerfilter angeboten.
Um diesen Filter nutzen zu können hinterlegen sie zuerst die notwendigen Herstellerlogos dieser [[fl:modul:herstellerlogos|Anleitung ]] gemäß.

|{{:fl:filterkonfiguration_bild_options.png| }}|

  * Darstellungsstil
     * Als Liste:           Nach jedem angezeigten Bild erfolgt ein Zeilenumbruch - Darstellung der Filterwerte vertikal
     * Aneinandergereiht:   Bilder werden nebeneinander angezeigt, automatischer Zeilenumbruch bei Filterbreite - Darstellung der Filterwerte horizontal
  * Bezeichnung anzeigen:   Den Namen des Filterwertes zusätzlich zum Bild anzeigen
  * Bildbreite:             Festgelegte Breite der Bilder in Pixel. (Bild wird skaliert)          
  * Bildhöhe:               Festgelegte Höhe der Bilder in Pixel. (Bild wird skaliert)

== Bereichsslider ==
Filterwerte im Schieberegler angezeigt.
Um diesen Filter auswählen zu können müssen für das entsprechende Attribut Zahlenwerte als Filterwerte hinterlegt sein.

|{{:fl:filterkonfiguration_bereich_options.png| }}|

  * Einheit: Einheit die zu den Zahlenwerten angezeigt werden soll.

== Weitere Optionen ==
Folgende weitere Optionen werden angezeigt, sofern der ausgewählte Filtertyp diese Option ermöglicht.

|{{:fl:filterkonfiguration_allgemein_options.png| }}|

  * CSS Klasse: Filter erhalten diese CSS Klasse zugewiesen. Ermöglicht die Umsetzung eigener CSS Anweisungen über Shopeigene Stylesheets
  * 'keine Filter verfügbar' Text: Dieser Text wird angezeigt sofern kein Filterwert verfügbar ist, der Filter aber als Hauptfilter definiert wurde. Wird dieses Feld leergelassen erscheint standardmäßig 'keine Filter verfügbar' bei leerem Filter.
  * Sortierung: Legt fest in welcher Reihenfolge Filterwerte angezeigt werden.
     * A-Z: Lexikographische Sortierung
     * Relevanz: Sortierung nach berechneter Relevanz der Ergebnisse
     * Häufigkeit: Die häufigsten Filterwerte werden zuerst gelistet
     * 0-9: Aufsteigende Sortierung
  * Häufigkeit anzeigen: Zeigt wenn aktiviert, die Anzahl der Artikel pro Filterwert in Klammer neben dem Filterwert an.
