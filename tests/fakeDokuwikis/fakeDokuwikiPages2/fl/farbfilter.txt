=====Farbfilter=====

Mit dem **Farbfilter** können Sie die Farbwerte Ihres Filters als Farbkacheln darstellen.\\

{{:fl:farbfiltervn2.png|Farbfilter Vorher/Nachher}}

Hier wird Ihnen Schritt für Schritt erklärt, was Sie tun müssen, 
um die Funktion in Ihren Shop einzubauen.

=====Einbindung der Codes=====

Das zu bearbeitete Template finden sie im FINDOLOGIC-Kundenaccout unter dem Menüpunkt ''Templatemanager''.
Fügen Sie folgenden Code ganz oben im Template ein. Dieser Code definiert das Aussehen der Farbkacheln Ihres Filters.
<code css>
{literal}
<style>
.colorPickerElementWrapper {
	position:relative;
	text-align:center;
	width:25px;
	height:25px;
}

.colorPicker {
	width:25px;
	height:25px;
	position:absolute;
	left:0; top:0;
}

.noColorFound {	
	padding-top:7px; 
} 
</style>
{/literal}
</code>

Etwas weiter unten im Template finden Sie eine Codestelle, in der die Filter definiert werden.
Fügen Sie hier folgenden Code ein:

<code>
<div class="flFilterBox">
<strong style="font-size:12px;">Farbe</strong>
<div class="flFilterBoxFilters">{filter name="Farbe" type="color-picker"}</div>
</div>
</code>

<note important>Bei ''filter name="Farbe"'' müssen Sie den Namen Ihres Filters eingeben, dieser ist von Shopsystem zu Shopsystem unterschiedlich.</note>

Das Ganze sollte dann in etwa so aussehen:

{{:fl:farbfilter_code.png|Die html2a.tpl}} 

=====Einstellen der Farben im Kundenaccount====

Im Kundenaccount unter "Einstellungen -> Farbfilter" können Sie nun Ihre Farben definieren.
Es gibt drei Möglichkeiten um Farben zu definieren:

    * Sie fügen einen eigenen Farbton (einfarbig) hinzu (z.B. ''#00DD00'')
    * Sie tragen den Pfad zu einem Bildverzeichnis auf Ihrem Server ein und laden entsprechend benannte Bilder hoch, die im Farbfilter angezeigt werden
    * Sie benutzen bereits vordefinierte Farben von FINDOLOGIC

<note important>Sollte anstatt einer Farbkachel ein "?" im Filter angezeigt werden, so liegt dies daran, dass die Farbe bisher noch nicht definiert wurde.</note>
<note important>Bitte beachten Sie während der folgenden Anleitung, dass die Farben, die Sie hier definieren, exakt gleich wie im Shop heißen müssen.</note>

==== Tab: Farbton ====

Im Farbton-Tab befindet sich ein Formular, mit dem Sie eigene Farbtöne hinzufügen können.
Hierzu muss ein Farbname (z.B. Königsblau) und ein HTML(Hex-)Code der Farbe eingetragen werden (z.B. #1100FF)

<note>Hier ein praktisches Tool, um HEX-Codes für Farben herauszufinden: http://html-color-codes.info/webfarben_hexcodes/ (externe Seite)</note>

{{:fl:produktfarben_tab_1.png|}}

=== Hinweise ===

    * Der selbst definierte Farbton überschreibt einen vordefinierten Farbton mit gleichem Farbnamen    
    * Nur ein gültiger Hexcode kann gespeichert werden
    * Über den Hexcode können nur einfärbige Farbdarstellungen umgesetzt werden

==== Tab: Farbbildverzeichnis ====

Um auch Bilder (z.B. Texturen) anstatt von Hexcode-Farben darstellen zu können, gibt es die Möglichkeit einen Pfad zu einem
Bilder-Verzeichnis anzugeben und Bilder für die Darstellung von Farbcodes zu verwenden.

{{:fl:produktfarben_tab_2.png|}}


Diese Bilder werden dann ergänzend zu den bereits definierten Farben dargestellt.


=== Dateinamen Konventionen ===

Zu den Dateinamen der Bilder gibt es Folgendes zu beachten, nutzen Sie hierfür gleich unser Dateinamen-Tool unter [[https://secure.findologic.com/login/index.php?page=filter_base_url|Link]]:

  * Alle Sonderzeichen (inkl. Leerzeichen) werden durch "_" ersetzt
  * Mehrere aufeinanderfolgende "_" werden anschließend durch einen "_" ersetzt.
  * Es werden nur Kleinbuchstaben verwendet
  * Umlaute werden durch ihre Grundform ersetzt: ä = a, ö = o, ü = u, ß = s
  * Per default werden ".jpg" Dateien verwendet

Hier ein paar Beispiele:
^Farbname^ ^Dateiname^
|"Rot"|→|"rot.jpg"|
|"Schwarz Weiss"|→|"schwarz_weiss.jpg"|
|"Grün"|→|"grun.jpg"|

==== Tab: Vordefinierte Farben ====

FINDOLOGIC hat bereits einige Farben für Sie definiert.

Im Tab 'Vordefinierte Farben' auf der Produktfarben-Seite finden sie eine Liste mit allen vordefinierten Farben und entsprechenden HTML (Hex)-Code

{{:fl:produktfarbe_tab3.png|}}



==== Anzeigepriorität der Farben im Filter ====

Die Priorität wie die Farben angezeigt werden:

    * Bild
    * Selbst-definierter Farbton
    * Vordefinierter Farbton

Findet der Filter ein Bild zu einem Farbnamen, so wird das Bild dargestellt (vor dem selbst- und vor-definierten Farbton).

Existiert ein selbst-definierter Farbton, der den gleichen Namen besitzt wie ein vordefinierter Farbton, so wird Ihr Farbton den vordefinierten überschreiben.
