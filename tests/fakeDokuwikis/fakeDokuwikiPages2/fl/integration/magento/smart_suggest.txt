====== Integration der Smart Suggest bei Magento ======

  - Laden Sie im Kundenaccount den Smart Suggest-Proxy herunter
  - Laden Sie die Datei (z.B. per FTP) ins Hauptverzeichnis Ihres Shops hoch
  - Stellen Sie sicher, dass die Datei von außen erreichbar ist, indem Sie diese im Browser aufrufen. Liegt Ihr Shop z.B. unter http://www.mein-shop.de/ rufen Sie die Datei http://www.mein-shop.de/autocomplete.php auf.

Nun können die benötigen Skripte integriert werden:

    - Öffnen Sie im Backend Ihres Shops das Menü ''CMS > Statische-Blöcke''
    - Wählen Sie dort den Block ''Footer Links'' aus.
    - Auf der Bearbeiten-Seite klicken Sie nun auf den Button ''Editor anzeigen/ausblenden'' (sofern dieser nicht bereits ausgeblendet ist).
    - Im FINDOLOGIC-Kundenaccount finden Sie bei der Konfiguration der Smart Suggest ein Code-Schnipsel, der in Ihrem Shop einzufügen ist
    - Fügen Sie diesen an das Ende des Quellcodes ein
    - Klicken Sie rechts oben auf ''Block speichern'' um die Änderung zu übernehmen
