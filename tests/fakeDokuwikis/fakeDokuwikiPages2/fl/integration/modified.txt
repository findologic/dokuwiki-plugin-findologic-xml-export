===== FINDOLOGIC Integration für modified =====

==== Wichtige Hinweise ====

=== Kompatibilität ===

FINDOLOGIC wird bei modified Shops nur per Direct Integration unterstützt.

==== Einrichtung im Shopsystem Backend ====

  * Wählen Sie im Administrator Bereich den Menüpunkt ''Module -> System Module''
  * Sollten Sie eine Shopversion kleiner als 2.0 verwenden, wählen Sie stattdessen bitte ''Export Module''.

{{:fl:integration:modified_2.png|}}

  * Wählen Sie ''Aktion'' beim FINDOLOGIC Modul

{{:fl:integration:modified_3.png|}}

  * Tragen Sie hier Ihren Shopkey und Ihre Service URL ein und wählen ggf. persönliche Einstellungen. Shopkey und Service URL finden Sie im FINDOLOGIC Kundenaccount unter dem Menüpunkt ''Account -> Stammdaten''.
  * Bitte beachten Sie, dass für die Verwendung einer weiteren Sprache ein weiterer FINDOLOGIC Service notwendig ist.

{{:fl:integration:modified_4.png|}}

==== Einrichtung im FINDOLOGIC Kundenaccount ====

Sollten Sie ein Basic Paket oder höher verwenden, können Sie die FINDOLOGIC-Smart Suggest verwenden.

  * Gehen Sie bitte hierzu im FINDOLOGIC Kundenaccount auf den Menüpunkt ''Einstellungen -> Smart Suggest'' und speichern zumindest einmal Ihre gewünschte Konfiguration.

Die Einstellungen sind mit sofortiger Wirkung aktiv. Um dies zu prüfen kann es vorkommen, dass Sie zunächst Ihren Browser Cache leeren müssen.

==== Design Anpassung ====

  * Für die Darstellung der Ergebnisse wird Ihr shopinternes Design verwendet. Filter, Sortierungsauswahl, etc. können Sie mit dem [[https://secure.findologic.com/dokuwiki/doku.php?id=fl:template_manager_2_0||FINDOLOGIC Template Manager]] direkt in Ihrem Kundenaccount bearbeiten.
