====== Einrichtung des Moduls ======

Sollte Ihre derzeitige Version der Shopsoftware nicht angeführt sein wenden Sie sich bitte direkt an support@findologic.com.
Wir prüfen die Kompatibilität gerne auch für Ihre Version.

===== Integration =====



 - Laden Sie sich das FINDOLOGIC-Integrationpackage herunter: {{:fl:integration:oscommerce.zip|oscommerce.zip}}
  - Kopieren Sie den Inhalt der osCommerce.zip in das Hauptverzeichnes des Shopsystems:
  - Öffnen Sie die Konfigurationsdatei ''findologic_config.inc.php'', und tragen Sie unter den ersten drei Punkten den Shopkey, die Shop-URL und die Service-URL, welche Sie im Customer-Login finden, ein.
  * WICHTIG: Vergessen Sie bei den URLs nicht den Slash am Ende, da es sonst zu Problemen bei der Darstellung der Ergebnisse kommt.
  * Beispiel: 
              
              define("FL_SHOP_ID", "1234567890ABCDEF1234567890ABCDEF");
              define("FL_SHOP_URL", "http://www.mein-shop.de/");
              define("FL_SERVICE_URL", "http://srv05.findologic.com/ps/mein-shop.de/");

  - Legen Sie im Hauptverzeichnis Ihres Shops ebenfalls den Ordner "export" an, falls dieser noch nicht existiert.
  - Bitte kontaktieren Sie uns und teilen Sie uns mit, dass die Daten nun indiziert werden können.
  - Wir kontaktieren Sie, sobald die Daten erfolgreich indiziert worden sind. Die Suche kann dann bereits verwendet werden.
  - Um das Modul zu aktivieren, müssen Sie das Ziel des Sucheingabefeldes von ''advanced_search_result.php'' auf ''findologic.php'' umstellen.

====== Konfiguration im FINDOLOGIC-Kundenbereich ======

Im [[https://secure.findologic.com/login/|Kundenbereich]] ist keine weitere Einrichtung erforderlich.



====== Erzeugen einer findologic.php aus einer bestehenden advanced_search_result.php ======
<note important>
Dieser Schritt ist nur notwendig, falls die Standard-Integration nicht funktioniert.
</note>

  - Kopie von advanced_search_result.php anlegen
  - Den Bereich zwischen <code>
<!-- body_text //-->
<!-- body_text_eof //-->
</code> ersetzen durch: <code>
<!-- body_text //-->
    <td width="100%" valign="top"><table border="0" width="100%" cellspacing="0" cellpadding="0">
      <tr>
        <td>
<?php
            echo $content;
?>
        </td>
      </tr>
    </table></td>
<!-- body_text_eof //-->
</code> Am Ende der Kopie folgendes anfügen: <code>
<?php
        }
    }
?></code>
  - Den gesamten Code der findologic.php bis zu <code><!doctype html public "-//W3C//DTD HTML 4.01 Transitional//EN"></code> kopieren und den Bereich über <code><!doctype html public "-//W3C//DTD HTML 4.01 Transitional//EN"></code> in der Kopie ersetzen.
  - Die Kopie in findologic.php umbenennen.

===== Design Anpassung =====

Die CSS-Datei zur Darstellung der Filter und Ergebnisse wird automatisch in das FINDOLOGIC-Template eingebunden, Sie können auf dieses im FINDOLOGIC-Template-Manager zugreifen. Die URL der CSS-Datei lautet: ''http://srvXY.findologic.com/ps/ihre-domain.de/css/findologic.css''.
Wenn Sie diese anpassen wollen, kopieren Sie die Datei in Ihren Shop und binden Sie die modifizierte CSS-Datei dort ein. 

<note important>Vergessen Sie nicht den Original-Verweis im FINDOLOGIC-Template zu entfernen.</note>

Im Stylesheet finden Sie einen Block mit der Bezeichnung: **#flContentContainer**.
Dort können Sie den **border** und die **margin-right** auskommentieren (mit 2 Slashes).
Somit gibt es keine Größenveränderung im Ergebnis-Container.

Um die Ergebnisanzeige vorab bereits farblich etwas anzupassen, finden Sie hier 3 wichtige Schritte: [[fl:integration_layout]].


Die Templates zur Darstellung des Filters und der Ergebnisse können Sie im Kundenbereich anpassen.

Dokumentation zu den Templates finden Sie unter: [[fl:template_manager]] bzw. [[fl:template_manager_2_0]] und [[fl:template_manager:plugins]].

===== Falsche Produkt-URLs in den Suchergebnissen =====

Falls in den Suchergebnissen falsche Produkt-URLs benutzt werden, können Sie das Format in der Datei ''findologic_config.inc.php'' in der Funktion ''get_url()'' anpassen.
