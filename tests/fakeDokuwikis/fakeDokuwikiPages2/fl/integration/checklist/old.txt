Diese Seite dient dazu eine Übersicht zu geben über spezielle Angelegenheiten die nach der Integration von FINDOLOGIC auftreten können.

Wenn Sie alle Fragen dieser Checklist mit "JA" beantworten können nutzen Sie FINDOLOGIC auf einem bestmöglichen Weg.

Mit einer sehr hohen Wahrscheinlichkeit können Sie bereits alle Fragen mit "JA" beantworten, die Checkliste dient dazu Fehler schnellstmöglich zu erkennen, damit diese auch rechtzeitig behoben werden können!

Im Anschluss finden Sie Lösungsvorschläge bzw. bei welchen Problemen wir Sie bitten sich direkt an uns zu wenden.

  * Erscheinen die Suchergebnisse nach abschicken der Suchanfrage?
  * Erscheinen Filterboxen über den Suchergebnissen?
  * Wird das FINDOLOGIC Logo oberhalb der Suchergebnisse angezeigt?
  * Wurde das findologic.css File eingebunden?
  * Werden die Filterboxen so angezeigt wie es Ihrem Layoutwunsch entspricht?
  * Werden die Ergebnisse so angezeigt wie es Ihrem Layoutwunsch entspricht?
  * Werden alle Preise korrekt angezeigt?
  * Finden Sie die Produkte die Sie bei einer von Ihnen gestellten Abfrage erwarten?
  * Stimmen die vorgegebenen Preisbereiche der Preisfilter mit den dargestellten Preisen überein?
  * Funktioniert die Verlinkung der Ergebnisse zu den Produkten?
  * Funktionieren die Links der Filtermöglichkeiten?
  * Werden Umlaute und Sonderzeichen korrekt dargestellt?
