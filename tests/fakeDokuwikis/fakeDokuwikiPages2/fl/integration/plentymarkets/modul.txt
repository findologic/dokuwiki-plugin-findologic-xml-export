====== FINDOLOGIC Integration für plentymarkets ======

===== plentymarkets Backend =====
Öffnen Sie das Backend Ihres Shops und loggen Sie sich ein.

==== API-Benutzer erstellen ====
  - Wählen Sie im Menü ''Einstellungen -> Benutzerkonten''
  - Wählen Sie dort ''Benutzer hinzufügen''
  - Weisen Sie dem Benutzer die Klasse ''API'' zu
  - Speichern Sie die Einstellungen

==== Konfigurieren ====

  - Wählen Sie im Menü ''Märkte -> Webshop -> Einstellungen'' 
  - Öffnen Sie die Einstellungen unter ''FINDOLOGIC''
  - Sollten hier noch keine Konfigurationsmöglichkeiten verfügbar sein, fügen Sie unter ''Sprache hinzufügen'' die jeweilige Sprache Ihres Webshops hinzu
  - Wählen Sie für die zu konfigurierende Sprache folgende Einstellungen:
    * **FINDOLOGIC für die Webshop-Suche verwenden**: Ja (sofern Sie FINDOLOGIC in der Suche verwenden wollen)
    * **FINDOLOGIC für die Navigation benutzen**: Ja (sofern Sie FINDOLOGIC in der Navigation verwenden wollen)
    * **Identifikationsschlüssel**: 32-stelliger Shopkey, dieser steht oberhalb als ''Shopkey''
    * **FINDOLOGIC-URL**: Die URL des FINDOLOGIC-Service, diese steht oberhalb als ''FINDOLOGIC/Service-URL''
    * **Artikelanzahl pro Seite**: Wählen Sie, wie viele Artikel pro Seite ausgegeben werden sollen
  - Speichern Sie die Einstellungen
{{:fl:integration:plentymarkets2.png|}}
	

===== FINDOLOGIC-Integrationsassistent=====
Tragen Sie nun unterhalb folgendes ein:
  * **CSV-URL**: Hier ist kein Eintrag erforderlich
  * **Export-URL**: Hier ist kein Eintrag erforderlich
  * **Export-ID**: Benutzernamen des soeben angelegten API-Benutzers aus dem plentymarkets-Backend
  * **Export-Passwort**: Passwort des soeben angelegten API-Benutzers aus dem plentymarkets-Backend

Klicken Sie anschliessend auf ''Modul überprüfen''
