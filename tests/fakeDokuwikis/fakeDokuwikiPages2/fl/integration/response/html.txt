====== Response: HTML =====
Wie das Ergebnis aussehen soll kann über ein Template-System gesteuert werden.
Sie können aus folgenden drei Varianten wählen:

===== Ausgabe  =====

Hierbei generiert unser Service ein HTML-Fragment, welches anstelle der bisherigen Suchergebnisse direkt in den Shop eingebunden werden kann.
Das Fragment enthält bereits fertige Darstellungen für die Filter und Elemente zur Seitennavigation.

Die Suchergebnisse werden lediglich als strukturierte Produkt-IDs zurückgeleifert.\\
Bevor das HTML angezeigt wird, müssen die IDs vom Modul geparsed werden und über ein shopinternes Listing gerendert werden.\\
Dieses Verfahren stellt sicher, dass die Darstellung der Produkte konsistent mit dem Rest des Shops ist und aktuelle Daten wie Lieferstatus direkt aus dem Shop ausgelesen werden können.

Layout und Design können völlig frei über den [[fl:template_manager_2_0|Template-Manager]] bei FINDOLOGIC eingestellt werden.

Die Ausgabe der Artikel IDs erfolgt in definierten Tags wie folgt:

    <div id="flResults">
        
        <flProductID>1</flProductID><flProductID>2</flProductID><flProductID>3</flProductID>
    </div>
