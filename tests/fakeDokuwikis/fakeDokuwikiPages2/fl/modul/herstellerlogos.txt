===== Herstellerlogos anzeigen =====

{{:fl:template_manager:hersteller_doku.png|}}

Sie haben die Möglichkeit die Hersteller innerhalb des Filters inkl. Logo anzeigen lassen. Dazu wurde das Plugin für den bisherigen Herstellerfilter erweitert. Sie haben dazu die Möglichkeit die Anzeige individuell zu gestalten wie im Folgenden erläutert wird. Beachten Sie ebenfalls die allgemeine Dokumentation zu unseren Plugis unter: [[fl:template_manager:plugins]]
Bei entsprechender Konfiguration werden die hier angegebenen Hersteller-Logos auch in der [[fl:integration:smart_suggest|Smart Suggest]] ausgegeben. 

Wir empfehlen zur Einbindung folgende Reihenfolge:

  - Loggen Sie sich in unserem Kundenbereich ein und legen Sie unter [[https://secure.findologic.com/login.symfony/web/app.php/shop/logos|Link]] als "Pfad zum Verzeichnis" die Adresse an unter welcher Sie Ihre Hersteller Logos hinterlegen
  - Legen Sie unter diesem Verzeichnis ein Unteverzeichnis namens "vendor" an
  - Bevor Sie die Logos anlegen beachten Sie bitte unbedingt die Datainamen Konventionen die unter [[https://secure.findologic.com/login/index.php?page=filter_base_url|Link]] und im Folgenden erläutert werden.
  - Gehen Sie die Dokumentation durch und entscheiden Sie welches Format Sie wählen
  - Sammeln Sie Ihre Logos zusammen und legen Sie diese im gewünschten Format **im Unterverzeichnis "vendor"** an
  - Ändern Sie den Herstellerfilter in der [[https://secure.findologic.com/login.symfony/web/app.php/shop/filter|FINDOLOGIC Filterkonfiguration]] auf den Typ "Bild" ([[fl:template_manager:filter_konfiguration#filter_konfigurieren|weitere Informationen zur Filterkonfiguration]]). Sollten Sie ein Template ohne "{generated_filters}" Plugin nutzen, passen Sie die Parameter des Filterplugins im Templatemanager entsprechend an (siehe [[fl:template_manager:plugins#anzeige_der_herstellerlogos|Parameter]]).
  - Prüfen Sie das Ergebnis direkt in Ihrem Shop

==== Dateinamen Konventionen ====
Zu den Dateinamen der Bilder gibt es Folgendes zu beachten, nutzen Sie hierfür gleich unser Dateinamen-Tool unter [[https://secure.findologic.com/login.symfony/web/app.php/shop/logos|Link]]:

  * Alle Sonderzeichen (inkl. Leerzeichen) werden durch "_" ersetzt
  * Mehrere aufeinanderfolgende "_" werden anschließend durch einen "_" ersetzt.
  * Es werden nur Kleinbuchstaben verwendet
  * Umlaute werden durch ihre Grundform ersetzt: ä = a, ö = o, ü = u, ß = s
  * Per default werden ".jpg" Dateien verwendet. Sie können die Endung nach belieben bei dem Hersteller-Plugin verändern, die gewählte Dateiendung wird für alle Herstellerlogos verwendet!

Hier ein paar Beispiele:
^Herstellername^ ^Dateiname^
|"Philips"|→|"philips.jpg"|
|"The North Face"|→|"the_north_face.jpg"|
|"Häagen Dazs"|→|"haagen_dazs.jpg"|
|"D D V - Früchte-Handels GmbH"|→|"d_d_v_fruchte_handels_gmbh.jpg"|

==== Parameter ====

Zusätzlich zu den bisherigen Parametern des Plugins gibt es folgende weitere Parameter:

^ Parameter ^ Wert ^ Beschreibung ^ Default ^
| type | image-filter | Geben Sie diesen Parameter und Wert an um den Herstellerfilter mit Herstellerlogos anzugzeigen | |
| style | float | Liefert die Ergebnisse anstatt in einer vertikalen Liste, direkt anneinander gereiht (siehe Beispiele) | list |
| style | list | Liefert die Ergebnisse wie gewohnt in einer Listenstruktur | list |
| show_titles | //Integer// | Gibt an ob die Herstellerbezeichnungen ebenfalls angezeigt werden sollen | 0 |
| img_width | //Integer// | Setzt eine fixe Breite der Logos | |
| img_height | //Integer// | Setzt eine fixe Höhe der Logos | |
| img_filetype | //String// | Legt die Dateiendung aller Logos fest (ohne Punkt Bsp: "png", "gif") | "jpg" |

==== Hinweise, Tipps und Tricks ====
  * Die Dateiendung die Sie mitangeben bezieht sich auf ALLE Bilder!
  * Entscheiden sie früh welche Höhe oder Breite Sie wählen und führen ein Resampling der Bilder mit geeigneter Software durch.
  * Achten Sie bei einer "Float"-Ansicht darauf geeignete Logos zu verwenden um den Filter attraktiv zu halten.
  * Für jeden Herstellerfilter wird automatisch das Bild unter der angegebenen Base-URL aus dem Unterverzeichnis "/vendor" und der entsprechenden Bezeichnung verwendet.
  * Ein einfaches Beispiel: Für das Logo des Herstellers Adidas wird bei folgender unterstrichen angegebener Image Base-URL folgende img-url vewendet:\\
    "__<nowiki>http://www.meinshop.at/findologic_filterimages</nowiki>__/vendor/adidas.jpg"
